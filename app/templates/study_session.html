Hello Alex! Let us dive back into your calculus review. Specifically, we will look at the fundamental theorem of calculus. It connects differentiation and integration in a beautiful way:

$$\int_{a}^{b} f(x) dx = F(b) - F(a)$$

To see how this looks in practice, we can implement a simple numeric integrator in Python using the trapezoidal rule. Check this out:

python content_copy 
def trapezoidal_rule(f, a, b, n):
    h = (b - a) / n
    s = (f(a) + f(b)) / 2.0
    for i in range(1, n):
        s += f(a + i * h)
    return s * h
